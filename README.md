A simple todo list app with authentication
Tech stack :-
Frontend - React.js ,Axios
Backend - Node.js ,Express.js
Database - PostgreSQL
Authentication - JWT ,Bcrypt

   Setup Instructions

 1.Clone the repository
  git clone https://github.com/Abhix03x/todolist.git
  cd todolist
2.Back-end setup
  cd server
  npm install
  nodemon server.js
3.Front-end setup
  cd client
  npm install
  npm run dev
